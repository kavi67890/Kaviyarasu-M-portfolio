# Kaviyarasu M - Personal Portfolio

A professional, interactive personal portfolio website showcasing a unique intersection of **Finance, Healthcare, and Data Science**.

## 🚀 Features

- **Interactive 3D Background**: Custom animated background using [Three.js](https://threejs.org/) featuring floating particles and connecting molecular/data structures.
- **Modern UI/UX**: Dark-themed glassmorphism design with sleek animations, magnetic cursor glow, and scroll-reveal effects.
- **Fully Responsive**: Fluid layout ensuring a seamless experience across all devices (desktops, tablets, and smartphones) with an optimized mobile navigation menu.
- **Performance Optimized**: Lightweight and fast-loading built strictly with vanilla JavaScript and CSS.

## 🛠️ Technologies Used

- **HTML5** - Semantic structure
- **CSS3** - Custom styling, CSS root variables, Flexbox/Grid layouts, media queries, and keyframe animations
- **JavaScript (ES6+)** - DOM manipulation, form handling, intersection observers, and smooth scrolling
- **[Three.js](https://threejs.org/)** - WebGL library for rendering the interactive 3D background canvas
- **[FontAwesome](https://fontawesome.com/)** - Scalable vector icons

## 💻 How to Run

This is a completely static website with no build step required.

1. Clone or download the repository to your local machine.
2. Simply double-click the `index.html` file to open it in any modern web browser (Chrome, Firefox, Safari, Edge).
3. (Optional) Run it via a local live server like the VS Code "Live Server" extension for hot-reloading during development.
